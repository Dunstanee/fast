OBJECTIVES

The job duties of a computer programmer vary widely. Some programmers build websites, others create mobile apps, some write controller logic for appliances and other do data analysis. SAlaries for programmers also vary widely with the median at around $71,000 per year with education minimums from trade school certificate to Ph.D. In spite of these variations, the objectives of each job are quite similar. Specified requirements, delivered on time and on budget. No matter the language or tools, a programmer must write code that runs correctly. Unfortunately, we have all experienced poorly written software that pops up error messages or looks up computers. This is never acceptable. A good programmer knows how to write code incrementally, test each new feature and effectively handle exceptions.


A programmer's code must also make sense to other programmers. It must be well structured, annotated with comments and supplimented with good technical documentation. All code goes through periodic revisions and is often updated by others, so be kind to co-workers and write maintainable code . Many hours of development time can be saved when the original coder writes clean, readable code.
